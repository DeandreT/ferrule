Wire any data
into any other data.

ferrule is a graphical any-to-any data mapping tool — draw wires from a source schema to a target schema, drop in functions, filters, aggregates and group-bys, then run it anywhere: a native GUI, a headless CLI, or your browser.

XML · JSON · CSV · SQLite · PDF sources · X12 & EDIFACT EDI — plus best-effort import and export of MapForce .mfd designs.

▶ try it in your browser read the docs source on GitHub

What's in the box

⚙ A real mapping engine

Repeating elements become loops; enclosing values broadcast into nested rows; filters, group-by, aggregates (count / sum / avg / min / max / join) and cross-source lookups are first-class. Absent data flows as null instead of crashing your run.

⇄ Any-to-any formats

XML (XSD-driven, attributes included), JSON (JSON Schema with $ref), CSV, SQLite tables, and configured visual PDF sources — mixed freely per side. Multiple named sources can feed one target.

⭑ EDI that actually parses

X12 and EDIFACT with composite elements, qualifier-driven loops (HIPAA HL hierarchies), repetition separators, and a lenient mode for slim schemas — validated against real 837 / ORDERS traffic.

🖱 Visual, headless, or embedded

Draw mappings in the egui-based GUI, run them with ferrule run in CI, or embed the pure-Rust engine — it compiles to WebAssembly (that's the live demo).

🧪 Warning, not wall

Importers never die on the first unsupported construct: they convert everything they can and hand you one actionable warning per skipped piece, so a partial import still beats redrawing a mapping.

🦀 Small, honest Rust

A workspace of focused crates around one format-agnostic IR. No services, no runtime, no telemetry. GPL-3.0 licensed.

Migrating from MapForce?

ferrule imports MapForce .mfd designs (best-effort), covering XML, JSON, CSV, FlexText, XLSX, visual PDF sources with multi-page table regions, vertical collages, and marker-delimited records, Protocol Buffers, single-table SQLite, common core functions, aggregates, joins, and controlled iteration. Supported project shapes also export back to .mfd; unsupported pieces remain explicit warnings you can finish in the editor.

# convert a design, then run it
cargo run -p cli -- import-mfd --mfd design.mfd --out project.json
cargo run -p cli -- run --project project.json --input orders.xml --output summary.json

MapForce is a trademark of its owner; ferrule is an independent project.

Get started

# clone & run the GUI
git clone https://github.com/DeandreT/ferrule
cd ferrule
cargo run -p gui

# or run a mapping headlessly — formats picked by extension
cargo run -p cli -- run --project examples/project.json \
    --input orders.xml --output order_lines.csv
