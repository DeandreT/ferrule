Chapter 5

MapForce import/export

ferrule provides clean-room, best-effort import and export for MapForce `.mfd` designs. Unsupported pieces become warnings where possible, so recoverable mappings stay editable and executable.

Import a design

cargo +nightly run -p cli -- import-mfd \
  --mfd design.mfd \
  --out project.json

Import resolves supported document components, schemas, graph nodes, target scopes, format options, paths, and warnings into a ferrule project. A design is rejected only when no usable source or target can be recovered.

Export a design

cargo +nightly run -p cli -- export-mfd \
  --project project.json \
  --out design.mfd

Export writes the representable project subset with generated schema, layout, or metadata siblings. Shapes that cannot be represented safely reject atomically instead of publishing partial wiring.

Representative coverage

  • XML, JSON, CSV, fixed-width, FlexText, XLSX, SQLite, EDI, Protocol Buffers, XBRL, HTTP XML, and visual PDF source components.
  • Core scalar functions, aggregates, lookups, joins, exceptions, generated sequences, value maps, grouping, sorting, and windows.
  • Named static sources, dynamic XML paths, independent targets, schema-equal copy-all scopes, generic XML element sequences, selected user-function shapes, and bounded scalar modules from adjacent XSLT, C#, Java, and XQuery source.
  • Expanded XML element and attribute names survive import and execution; foreign declarations export atomically as deterministic local XSD dependency files.
  • Structured XML string serializers preserve attributes, nested and repeated content, escaping, namespaces, and declaration policy, and export with generated XSD contracts.
  • Structured XML SQLite text columns and declared local relations remain executable and round-trip with their typed database connection metadata.
  • External EDI configuration directories and adjacent bounded ZIP packages compile into portable X12 or EDIFACT runtime schemas.
Interop is intentionally resilient. The importer favors partial executable recovery with actionable diagnostics over all-or-nothing conversion.
